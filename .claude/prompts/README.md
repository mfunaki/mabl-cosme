# Claude Code プロンプト集

このディレクトリには、Claude Code で使用するプロンプトテンプレートを格納します。

## 使い方

プロンプトファイルの内容をコピーして Claude Code に貼り付けるか、
ファイルを参照して指示を出してください。

```
@.claude/prompts/test-generation.md を使ってテストを生成して
```

## ファイル一覧

| ファイル | 用途 |
|---------|------|
| `test-generation.md` | テストコード生成用プロンプト |
| `code-review.md` | コードレビュー用プロンプト |
| `refactoring.md` | リファクタリング用プロンプト |
| `mabl-test.md` | mabl テスト作成用プロンプト |

## プロンプト作成のガイドライン

1. **明確な目的**: 何を達成したいかを明記
2. **コンテキスト**: 必要な背景情報を含める
3. **制約条件**: 守るべきルールを記載
4. **出力形式**: 期待する出力フォーマットを指定
